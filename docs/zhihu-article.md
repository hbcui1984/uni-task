## 为什么选择 uni-task？

### 1. 完全开源，永久免费

项目采用 MIT 协议开源，你可以：
- 免费用于个人或商业项目
- 自由修改和定制
- 不用担心哪天涨价或停服

**GitHub 地址**：https://github.com/hbcui1984/uni-task.git

### 2. 成本极低

来算一笔账：

| 方案 | 10 人团队年费用 |
|------|----------------|
| Tower 专业版 | ¥2,388/年 起 |
| Teambition 企业版 | ¥1,188/年 起 |
| **uni-task** | **¥0 ~ ¥50/年** |

uni-task 基于 uniCloud 云开发，使用阿里云或腾讯云的 Serverless 服务：
- **开发测试阶段**：完全免费
- **小团队日常使用**：基本在免费额度内
- **即使超出免费额度**：按量付费，一年可能也就几十块钱

> 对比一下：Tower 专业版 10 人团队一年 2388 元，uni-task 可能一年只要一杯奶茶钱。

### 3. 一套代码，全端运行

基于 uni-app 开发，**一套代码**搞定：
- H5 网页版
- 微信小程序
- 支付宝小程序
- iOS App
- Android App
- 甚至 HarmonyOS

团队成员用什么设备都能用，不用单独开发多个版本。

### 4. 功能够用，不臃肿

专注于任务管理的核心功能，不搞花里胡哨：

**项目管理**
- 创建、编辑、归档项目
- 成员管理（管理员/普通成员）
- 邀请码快速加入

**任务管理**
- 任务增删改查
- 状态流转（待处理 → 进行中 → 已完成）
- 优先级设置（高/中/低）

- 任务分组与**拖拽排序**
- 子任务支持
- 附件上传

**团队协作**
- 任务分配
- 操作日志自动记录
- 项目动态实时展示
- 「我的任务」个人视图

## 快速部署指南

整个部署过程大概 **10 分钟**搞定。

### Step 1: 准备工作

1. 安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)（DCloud 官方 IDE）
2. 注册 DCloud 开发者账号
3. 开通 uniCloud 服务（免费）

### Step 2: 获取代码

```bash
git clone https://github.com/hbcui1984/uni-task.git
cd uni-task
npm install
```

### Step 3: 配置云服务

1. 用 HBuilderX 打开项目
2. 右键 `uniCloud-aliyun` → 关联云服务空间
3. 上传云函数和初始化数据库

### Step 4: 运行

```bash
# H5 版本
npm run dev:h5

# 微信小程序
npm run dev:mp-weixin
```

搞定！

## Q&A

**Q: 需要什么技术基础？**
A: 会基本的前端开发即可。如果只是部署使用，跟着文档操作就行。

**Q: uniCloud 收费吗？**
A: 有免费额度，小团队日常使用基本够了。超出按量计费，价格很便宜。

**Q: 可以部署到自己的服务器吗？**
A: uni-task 基于 uniCloud，目前需要使用阿里云或腾讯云的云空间。但数据是在你自己的云空间里，完全可控。

**Q: 支持私有化部署吗？**
A: 支持，但需购买`uni云开发软件版`的商业授权，[详见](https://doc.dcloud.net.cn/uniCloud/software/price.html)。
